# Glossary of Philippine Election Terms
| Term | Definition |
|---|---|
| Barangay | The smallest administrative unit of the  Philippine Government.There are about 42,011 barangays across the philippines. |
| LGU | Local Government Unit. The collective term for  provinces, cities, municipalities, and barangays. |
| COMELEC | Commission on Elections. The independent constitutional body that enforces all  election laws and administers every national and local election. |
| Precinct | The smallest voting unit in the Philippines. Each precinct is assigned a  clustered machine which can handle up to a maximum of 1,000 voters. |
| ER | Election Returns. This is a document which records the vote counts from a  specific Vote-Counting Machine at the precinct level. |
| COC | Certificate of Candidacy. The formal document a person files to run for office. |
| Party List | A representation system that reserves seats in the House of Representatives  for marginalized and underrepresented sectors (farmers, workers, women, youth, etc.).  Voters cast a separate ballot for a party list group, not an individual. |
| OAV | Overseas Absentee Voting. The mechanism allowing Filipino citizens abroad to vote in national elections. |
| LAV | Local Absentee Voting. A mechanism which allows specific groups to vote outside of a precinct |
| BLGF | Bureau of Local Government Finance. The agency that oversees the fiscal affairs of all LGUs, including budget allocation  and expenditure reporting. |
| BARMM | Bangsamoro Autonomous Region in Muslim Mindanao. An autonomous political region in southwestern Mindanao and the  Sulu Archipelago, with its own parliament and governance structure. |
| Senator | A member of the Philippine Senate. Twenty-four senators serve six-year terms, with twelve seats contested every three years  in a national election. Every voter nationwide votes for up to twelve senators regardless of region. |
| Over Votes | Ballots where a voter selected more candidates than the allowed maximum for a position. These votes are nullified for that position. |
| Under Votes | Ballots where a voter selected fewer candidates than the allowed maximum.  These are valid as a voter may choose not to fill all twelve senate slots, for example. |
| Valid Votes | The total votes counted after removing over votes. |
| Valid Ballot | A ballot accepted by the machine, regardless of how many positions were voted on. |
| Obtained Votes | The total votes a candidate received across all precincts in a given scope. |
| Machine ID | The unique identifier for a vote-counting machine (VCM) assigned to a specific precinct. |
| Contest ID | An identifier grouping all candidates running for the same position in the same jurisdiction. |
